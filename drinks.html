If the width of the browser is lowered to the point that one row needs to stack on top of the other, the row containing the drink names (col-md) will break first. The reason for this is that the col-md class lacks the col-sm-2 class's level of responsiveness. Consequently, the beverages row will begin to stack before the letters row does when the viewport width drops.
